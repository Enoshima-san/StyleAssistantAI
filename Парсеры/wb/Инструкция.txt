1. pip install aiohttp schedule

2. Настройка конфигурации
Отредактируйте файл config.py:

Основные параметры:
python
SEARCH_QUERY = "футболка"  # Поисковый запрос

# Параметры парсинга
MAX_CONCURRENT_REQUESTS = 40  # Максимальное количество одновременных запросов(нужно тестировать на каждой системе)
REQUEST_DELAY = 0.05          # Задержка между запросами (секунды)

# Сохранение результатов
JSON_OUTPUT_FILE = "Результат парсинга.json"

# Параметры поиска
SEARCH_PAGES = "ALL"  # "ALL" - все страницы, или число

# Автоматизация
AUTO_RUN = True               # True - включить, False - выключить
RUN_INTERVAL_MINUTES = 60     # Интервал между запусками (минуты)
MAX_RUNS = None               # Макс. запусков (None - бесконечно)

3. Запуск парсера
Способ 1: Основной запуск
main.py
Способ 2: Использование планировщика
scheduler.py

4. Режимы работы
1. Одиночный запуск
Установите в config.py:
AUTO_RUN = False
2. Автоматический запуск
Установите в config.py:
AUTO_RUN = True
RUN_INTERVAL_MINUTES = 60    # Запуск каждый час
MAX_RUNS = 10                # Ограничить количество запусков None - без ограничений

5.Проверка доступности товаров
1)Создайте файл Артикулы.txt в папке проекта
2)Добавьте артикулы товаров (по одному в строке)
3)Запустите product_checker.py
